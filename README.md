# Capsules

Sentiment Vaults

## Instructions

### Requirements

1. Rust
2. Foundry - https://github.com/gakonst/foundry

---

### Cloning Repo

```bash
git clone --recurse-submodules git@github.com:sentimentxyz/capsules.git
```

---

### Building Contracts

```bash
forge build
```

---

#### Configuring mappings

```bash
forge config > remappings.txt
```
